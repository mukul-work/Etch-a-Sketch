# Etch-a-Sketch

## Overview
A browser-based drawing application that allows users to draw on a customizable grid using mouse interactions.
This project exists to practice DOM manipulation, event handling, and dynamic UI updates using vanilla JavaScript.

## Demo
- []Live URL:](https://mukul-work.github.io/Etch-a-Sketch/)

## Features
- Dynamic grid generation based on user input
- Click-and-drag drawing interaction
- Clear/reset functionality
- Responsive layout that adapts to screen size

## Tech Stack
- JavaScript
- HTML
- CSS

## What I Learned
- Manipulating the DOM efficiently using JavaScript
- Handling mouse events for interactive UI behavior
- Dynamically creating and updating elements based on user input
- Structuring JavaScript logic to keep UI and behavior manageable
